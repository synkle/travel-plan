## 🎯 왜 V-Class인가?

### 6인 가족 최적화
```yaml
좌석 구성:
  - 총 7좌석 (여유 1석)
  - 2열: 독립 시트 2개 (시니어 편함)
  - 3열: 벤치 시트 3개 (아이들 2 + 짐 1)

ISOFIX:
  - 2열 양쪽 ISOFIX 장착
  - 7살 쌍둥이 카시트 완벽 설치

공간:
  - 대형 트렁크 (6인 짐 8일분 여유)
  - 슬라이딩 도어 (아이들 승하차 안전)
```

### 장거리 편의성
```yaml
편안함:
  - 독립 시트 (시니어 허리 배려)
  - 3-Zone Climate Control (앞/중/뒤 개별 온도)
  - 조용한 실내 (아이들 낮잠)

주행 보조:
  - Adaptive Cruise Control (장거리 피로도 ↓)
  - Lane Keeping Assist (안전)
  - 파노라마 선루프 (아이들 즐거움)

실용성:
  - USB 충전 포트 여러 개
  - 컵 홀더 풍부
  - 접이식 테이블 (간식 먹기)
```

```yaml

  - 우측 통행 (한국과 반대)
  - 로터리: 안쪽 차량 우선
  - 고속도로 추월: 왼쪽 차선만
  - 주차: 청색 P 표지 따라가기

  - 시내: 50km/h
  - 일반 도로: 80-90km/h
  - 고속도로: 130km/h (프랑스), 무제한 (독일 일부)

겨울 운전:
  - Füssen 지역 눈 가능 (11월)
  - 스노우 타이어 권장 (렌터카에서 제공)
  - 천천히 운전
```

### 주차 전략
```yaml
Reims: 호텔 주차장 or 공영 주차장
Füssen: 호텔 주차장 (무료 많음)
Colmar: 구시가 외곽 주차장 (€10-15/day)
```
