## 🎯 샴페인 하우스란?

### 개념
```yaml
Champagne House:
  - 샴페인 생산 업체
  - 지하 셀러 (Cellar) 투어
  - 샴페인 제조 과정 설명
  - 시음 (Tasting)

특징:
  - 수백년 역사
  - 거대한 지하 동굴
  - 수백만 병 저장
  - 프랑스 문화유산
```

### Reims vs Épernay
```yaml
Reims (추천 ⭐):
  - Big Brands: Taittinger, Veuve Clicquot, Mumm
  - 도시 안에 위치
  - 접근성 좋음
  - 호텔에서 5-10min

Épernay:
  - Avenue de Champagne (유명한 거리)
  - Moët & Chandon 본사
  - Reims에서 30min 거리
  - 시간 부족 시 Skip
```

## 👨‍👩‍👧‍👦 가족 관리 팁

### 7살 쌍둥이
```yaml
준비:
  - 샴페인 셀러 = "와인 동굴"로 설명
  - 지하 탐험 느낌
  - 주스 마실 수 있어!

주의:
  - 지하 = 약간 추움 (10-12°C)
  - 겉옷 챙기기
  - 1시간 = 집중력 한계
  - 뛰어다니지 않기

흥미 유지:
  - "수백만 개 병이야!" 숫자 강조
  - 사진 찍기
  - 마지막에 주스 시음 기대감
```

### 장인어른 + 장모님
```yaml
포인트:
  - 샴페인 역사 설명 (관심 ↑)
  - 시음 시간 여유롭게
  - 사진 찍어드리기

주의:
  - 지하 계단 많음 (천천히)
  - 온도 낮음 (겉옷)
  - 서서 듣는 시간 있음

구매:
  - 투어 후 기념품 샵
  - 샴페인 구매 가능
  - 면세 (Tax Refund)
```
