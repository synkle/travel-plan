# ⛴️ Bateaux-Mouches Seine Cruise (Sunset)

## 개요

**위치**: Port de la Conférence (Pont de l'Alma 옆, 에펠탑에서 도보 10분)
**소요 시간**: 약 1시간 10분
**추천 시간**: 17:00 탑승 (11월 노을 + 야경 연결)
**참조 Day**: Day 8 (Paris Sightseeing)

**왜 17:00 슬롯인가?**
```yaml
11월 파리 일몰: 17:05~17:15
  - 17:00 탑승 → 황금빛 노을 + 야간 조명 전환을 한 번에 감상
  - 에펠탑 스파클 (20:00) 전 휴식 + 이동 시간 확보
  - 아이들 저녁 루틴(21:30 취침)과도 잘 맞음
```

## 🎯 액티비티 설명

```yaml

  1. Port de la Conférence 출발
  2. 노트르담 앞에서 회항
  3. 에펠탑 귀환

하이라이트:
  - 에펠탑, 오르세 미술관, 루브르, 노트르담 야경
  - 오디오 가이드 (14개 언어) → 한국어는 앱 다운로드 필요
  - 실내(유리) + 오픈 데크 모두 이용 가능
```

### 가족 포인트
```yaml
7살 쌍둥이:
  - 움직이지 않아도 파리 주요 랜드마크를 한 번에
  - 실내 좌석 = 추위 대비
  - 오디오 가이드와 퀴즈 카드 제공 (요청 시)

시니어 동행:
  - 좌석에서 편안히 감상
  - 화장실/매점 선내 구비

부부:
  - 노을 → 야경 전환 사진 스팟
  - 샴페인/핫초코 구입 가능 (€6-12)
```

## 🎫 티켓 & 예약

### 가격 (2025 기준)
```yaml
Adult (12+)   : €18
Child (4-11) : €8.50
Under 4      : 무료
6인 가족 합계: €89 (성인4 + 어린이2)
```

### 예약 방법
```yaml
1. www.bateaux-mouches.fr 접속
2. "Cruise - General" 선택
3. 날짜: 2025년 11월 28일
4. 시간: 17:00 출항 선택 (16:30 집결)
5. 인원 입력 후 결제 (카드)
6. QR 티켓 이메일 수령 → 모바일 제시 가능
```

### 좌석 팁
```yaml
- 선착순 좌석 배정 → 16:30 이전 도착 추천
- 오픈 데크: 사진 최적, 추위 대비 담요 제공 (요청)
- 실내석: 전면 유리, 히터 가동
```

## 🧭 접근 & 체크인

```yaml
호텔 (Bastille) → 메트로 Line 1
  - Bastille → Franklin D. Roosevelt (환승)
  - Line 9 → Alma-Marceau 하차 (총 25min)

Alma-Marceau 역 → 선착장
  - 도보 5min (Pont de l'Alma 방향)
  - 알록달록한 보트 + "Compagnie des Bateaux-Mouches" 간판 확인

체크인:
  - E-ticket 스캔 후 진입
  - 탑승 전 단체 사진 (구매 선택)
```
