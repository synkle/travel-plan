# 🍷 Alsace Wine Route (알자스 와인 루트)

## 개요

**위치**: Alsace Region, France (Colmar 주변)
**일정**: Day 5 (2025.11.25 화) 종일
**목적**: 전통 마을 탐방 + 와인 문화 체험 + 가족 여유

**왜 중요한가?**:
```yaml
Colmar 2박 활용:
  - 여러 마을 방문 (차량 이동)
  - 전통 알자스 분위기
  - 어른들 = 와인 시음
  - 7살 쌍둥이 = 동화 마을 산책
  - 시니어 = 여유로운 페이스
```

```yaml
Route des Vins d'Alsace:
  - 총 길이: 170km (Strasbourg → Mulhouse)
  - 마을 개수: 70개 이상
  - 특징: 중세 마을, 하프팀버 건물, 와인 셀러
  - 와인: Riesling, Gewürztraminer, Pinot Gris

Colmar 주변 추천 마을:
  1. Riquewihr (리크비르) ⭐⭐⭐⭐⭐
```

## 🏘️ 마을별 가이드

### Riquewihr (리크비르) ⭐⭐⭐⭐⭐

```yaml
위치: Colmar에서 15km (차량 20min)
특징:
  - "프랑스에서 가장 아름다운 마을" 선정
  - 중세 성벽 (완벽 보존)
  - 하프팀버 건물 (사진 예쁨)
  - 와인 셀러 (Cave) 많음

Must-Do:
  ✅ Dolder Gate (중세 성문) 사진
  ✅ Grand Rue (메인 거리) 산책
  ✅ Wine Cellar 방문 (Hugel, Dopff)
  ✅ 와인 시음 (어른들)
  ✅ 기념품 가게 (알자스 특산품)

소요 시간: 2.5h (여유롭게)

주차:
  - 마을 입구 주차장 (€3-5/day)
  - V-Class OK
  - 도보 5min → 구시가

와인 시음:
  - Cave Hugel ⭐⭐⭐⭐⭐
    * 주소: 3 Rue de la 1ère Armée
    * 시간: 10:00-12:00, 14:00-18:00
    * 비용: €5-10/인 (3-4종 시음)
    * 어른 4인 = €20-40

  - Cave Dopff
    * 주소: 68340 Riquewihr
    * 시음 + 구매 가능

Best Time: 09:30-12:00 (사람 적음)
```

---

### Eguisheim (에기샴) ⭐⭐⭐⭐⭐

```yaml
위치: Colmar에서 10km (차량 15min)
특징:
  - 원형 마을 (Circular village)
  - 성을 중심으로 동심원 구조
  - 꽃으로 장식된 하프팀버 건물
  - "프랑스에서 가장 아름다운 마을" 선정

Must-Do:
  ✅ Grand Rue 원형 산책 (30min)
  ✅ 꽃 장식 건물 사진
  ✅ 성 광장 (Place du Château)
  ✅ 카페에서 Kougelhopf (알자스 빵) + 커피
  ✅ 와인 시음 (옵션)

소요 시간: 3h

주차:
  - 마을 외곽 주차장 (FREE!)
  - V-Class OK
  - 도보 5min → 구시가

와인 시음:
  - Cave Léon Baur
    * 주소: 6 Rue des Trois Pierres
    * 시간: 10:00-12:00, 14:00-18:00
    * 비용: €5-10/인

카페 추천:
  - Pâtisserie Gilg
    * 전통 알자스 빵 (Kougelhopf, Bretzel)
    * 커피, 핫초코 (아이들)
    * 6인 = €20-30

Best Time: 13:00-16:00 (점심 후 여유)
```

---

### Kaysersberg (카이저스베르크) - 옵션

```yaml
  - €5-10/인 (3-4종)
  - 어른 4인 = €20-40 (Riquewihr)

구매:
  - 와인 1-2병 (€15-30/병)

  - 기념품 or 선물
```

```yaml
시음 중 (30-45min):

  - 와인 병 구경 (흥미)

주의:

  - 30분 = 집중력 한계
  - 뛰어다니지 않기
```

## 👨‍👩‍👧‍👦 가족 관리 팁

### 7살 쌍둥이
```yaml
준비:
  - "동화 마을 간다!" 사전 설명
  - 하프팀버 건물 = "과자 집" 같다고
  - 사진 찍기 놀이

흥미 유지:
  - 마을 산책 = 보물 찾기 게임
  - 꽃 장식 건물 세기
  - 간식 자주 (에너지)
  - 와인 셀러 = "지하 동굴 탐험"

주의:
  - 돌길 (조심히 걷기)
  - 손 꼭 잡기
  - 화장실 자주 (30min마다)
```

### 장인어른 + 장모님
```yaml
포인트:
  - 와인 시음 = 즐거움
  - 전통 마을 = 감상
  - 사진 자주 찍어드리기

주의:
  - 걷기 많음 (편한 신발)
  - 돌길 = 천천히
  - 휴식 자주 (카페)
  - 무리하지 않기 (2개 마을 충분)

와인 구매:
  - 기념품 or 선물
  - 면세 (Tax Refund)
```
