#### 09:30 - 마리엔 다리 (Marienbrücke) ⚠️ 11월 결빙 폐쇄 가능
```yaml
⭐ Plan A: Marienbrücke (마리엔 다리) - 최고의 뷰포인트
위치: 성 위쪽 (도보 10min)
목적: 성 전경 사진 (포스터 각도!)

활동:
  ✅ 가족 단체 사진 (필수!)
  ✅ 성 배경 (디즈니 성 모델)
  ✅ 알프스 경치

주의:
  - 다리 = 좁고 높음 (아이들 손 잡기)
  - 바람 (목도리, 모자 날아감 주의)
  - 11월 = 사람 적음 (사진 예쁘게)
  - ⚠️ 11월 결빙 시 폐쇄 (안전 이유)

소요: 20-30min

⚠️ Marienbrücke 폐쇄 시 대체 뷰포인트:

Plan B - Jugend Viewpoint (유겐트 전망대):
  - 위치: 호엔슈반가우 마을 남쪽 (도보 15min)
  - 각도: 측면에서 성 전체 조망
  - 장점: 폐쇄 없음, 가족사진 배경 좋음
  - 접근: Hohenschwangau 주차장 → 도보 경사로
  - 소요: 왕복 30min

Plan C - Tegelberg 케이블카 전망대:
  - 위치: Tegelberg 산 정상 (케이블카 10min)
  - 각도: 멀리서 성 + 알프스 파노라마
  - 비용: €33/인 (왕복), 6인 = €198
  - 장점: 알프스 전망 최고, 케이블카 체험
  - 단점: 비용, 시간 +1h
  - 권장: 시간 여유 있으면

Plan D - Hohenschwangau 성 앞 호수:
  - 위치: Alpsee 호숫가 (주차장 5min)
  - 각도: 멀리서 노이슈반슈타인 + 호수 반영
  - 장점: 무료, 쉬움, 가족산책 겸
  - 단점: 거리 멀어서 성 작게 보임
  - 소요: 15-20min

현장 판단:
  - 티켓 부스에서 Marienbrücke 상태 확인
  - 폐쇄 시 즉시 Plan B (Jugend) 실행
  - 시간 여유 있으면 Plan C (Tegelberg) 추천
```
